Welcome

What you can see here is the default NagUI Testing and Example GUI. NagUI enables you to create your own GUIs based on HTML and Smarty. NagUI can also be used as Backend for toolbars an programs cause it also can deliver JSON data.

Another included GUI in NagUI is the Mobile GUI. You can set the GUI to Mobile for a list of users or in general. You can also test the Mobile GUI with adding the GET Parameter template=mobile to an url.

The newest Plugin is the Reporting Plugin. It can generate Reports using multiple Database Tables. A import script for Logfiles into Database is available. Reporting using the Livestatus Backend didnt work yet.
